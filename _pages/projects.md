## Research Projects

### Quantum Machine Learning for Autonomous Driving

**With Hyundai Motor Group**, 2023-2024

- Investigated quantum machine learning approaches for object detection in autonomous driving.
- Built simulation pipelines with Qiskit and PyTorch for hybrid quantum-classical models.

### Quantum Computer Architecture Design

**2025**

- Designed a high-level trapped-ion quantum computing architecture.
- Covered compilation, scheduling, and data pre/post-processing modules.
- Developed a Python library for integration with Qiskit-based execution workflows.

### Trapped-Ion Experiment Software Development

**2025-Present**

- Developed software for end-to-end control of trapped-ion experiments.
- Supports remote entanglement experiments and surface-trap ion systems in current lab operations.
